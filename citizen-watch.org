* 1. Descripción

Plataforma independiente de inteligencia ciudadana OSINT para España.
Agrega datos públicos oficiales sobre las principales preocupaciones
de la ciudadanía española.

URL local: http://192.168.1.147:8502

* 2. Stack técnico

- Python 3.13 + Streamlit 1.56
- SQLite (citizen_data.db) — sin dependencias cloud
- Plotly 6.6 — gráficos ligeros para Odroid C2
- BeautifulSoup4 + Requests — scrapers
- Odroid C2 / DietPi — nodo local 192.168.1.147
- Git + GitHub — control de versiones

* 5. Automatización

#+BEGIN_SRC bash
# Cron en Odroid C2 — 06:08 cada día
08 06 * * * /bin/bash /home/dietpi/dashboard_citizen/scripts/push_data.sh

# Arranque manual
cd ~/dashboard_citizen
source venv/bin/activate
streamlit run app.py --server.port 8502 --server.address 0.0.0.0

# Log
tail -f ~/dashboard_citizen/logs/cron.log
#+END_SRC

* 8. Notas migración Streamlit Cloud

La DB SQLite no persiste en Streamlit Cloud.
Opciones para despliegue público:
1. Render + disco persistente (recomendado, ya tenemos experiencia)
2. CSV estáticos en repo (más simple, menos potente)
3. TinyDB / JSON files versionados en git
